Fixed-position-out-of-view-scroll-prepaint

Decription:This test is applicable on platforms that do accelerated compositing, can prepaint out-of-view contents and can scroll the contents into the viewport without repainting them.

The following settings need to be enabled:

- acceleratedCompositingForFixedPositionEnabled
- fixedPositionCreatesStackingContext

STEPS TO TEST: To test, scroll the page up and down.

TEST PASS: The test passes if there is no temporary or permanent red in the page.
